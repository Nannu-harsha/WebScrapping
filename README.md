# WebScrapping
This project demonstrates basic CRUD operations (Create, Read, Update, Delete) using Python and the Pandas library.

Features
Create new records
Read/display data
Update existing values
Delete rows and columns

Run the Project
pip install pandas
python main.py

Output
Displays updated DataFrame after create & update
Shows final DataFrame after delete operations

Purpose
Beginner-friendly project to understand data manipulation using Pandas.
